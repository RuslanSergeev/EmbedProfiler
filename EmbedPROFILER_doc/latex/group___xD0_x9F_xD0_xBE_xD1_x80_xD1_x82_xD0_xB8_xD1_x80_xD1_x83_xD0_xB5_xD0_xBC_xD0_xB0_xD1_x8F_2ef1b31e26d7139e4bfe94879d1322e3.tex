\subsection{Подробное описание}
Эти функции необходимо определить для вашей платформы. Остальная часть кода профилировщика является платформо-\/независимой. \-Embed\-P\-R\-O\-F\-I\-L\-E\-R проектировался для исполнения во встраиваемых системах независимо от аппаратной платформы. Для того, чтобы \-Embed\-P\-R\-O\-F\-I\-L\-E\-R можно было перенести на интересующую вас платформу, необходимо реализовать на данной платформе определённые в этой секции процедуры и макросы.
