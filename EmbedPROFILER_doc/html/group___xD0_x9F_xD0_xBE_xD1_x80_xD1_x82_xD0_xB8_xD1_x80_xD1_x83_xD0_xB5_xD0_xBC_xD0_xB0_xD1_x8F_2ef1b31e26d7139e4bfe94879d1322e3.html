Подробное описание

Эти функции необходимо определить для вашей платформы. Остальная часть кода профилировщика является платформо-независимой.

EmbedPROFILER проектировался для исполнения во встраиваемых системах независимо от аппаратной платформы. Для того, чтобы EmbedPROFILER можно было перенести на интересующую вас платформу, необходимо реализовать на данной платформе определённые в этой секции процедуры и макросы.

Макросы

#define PROFILER_global_stop ( )    PROFILER_local_stop()

Окончание работы глобального профилировщика.

Запоминается время окончания работы системы профилирования. Которое используется в дальнейшем для расчёта времени работы системы профилирования.Добавлено в портируемый раздел, поскольку завершение работы профилировщика может быть платформо-зависимой.

См. определение в файле PROFILING_tools.h строка 256

#define PROFILER_TOTAL_PROFILERS   1024

Максимальное число работающих профилировщиков времени вызова.

Это число определяет размеры массивов для хранения указателей на названия функций, имена файлов, номера строк и число вызовов.

void PROFILER_log ( )

Вывести отчёт о работе профилировщика.

Сообщает затраченное на работу каждого профилировщика время, имена файлов, номера строк, названия функций где был создан данный профилировщик. Также сообщает число вызовов данного профилировщика.

PROFILER_timestruct  PROFILER_end_time,
int32_t  PROFILER_instance_index 
)

Обновить общее время затраченное на работу указанного профилировщика за все его вызовы.

Добавляет время потраченное указанным профилировщиком за последний вызов к общему времени за все вызовы, затраченному указанным профилировщиком.

Аргументы:
[in]PROFILER_timestructPROFILER_begin_time
Время начала работы (очередного вызова) локального профилировщика.
[in]PROFILER_timestructPROFILER_end_time
Время окончания работы (очередного завершения) локального профилировщика.
[in]int32_tPROFILER_instance_index
Индекс локального профилировщика.
